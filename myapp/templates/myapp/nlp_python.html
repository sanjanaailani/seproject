1. Variables & Data Types

A variable is like a labeled box to store information in your program.

# Example variables
text = "Natural Language Processing"
number = 10
is_fun = True
height = 5.8
items = ["NLP", "ML", "AI"]
user = {"name": "Ava", "age": 20}
  • int: whole numbers like 3
  • float: decimal numbers like 3.14
  • str: text values like "hello"
  • bool: true/false values
  • list: group of items
  • dict: data in key-value pairs

2. Strings & Lists

Strings are sequences of characters. Lists are sequences of anything!

# String methods
sentence = "I love NLP"
print(sentence.upper())  # I LOVE NLP
print(sentence.split())  # ['I', 'love', 'NLP']

# List operations
apps = ["chatbots", "translators", "summarizers"]
apps.append("voice assistants")
print(apps[0])  # chatbots

3. Loops & Conditions

Use if, else, for and while to make decisions and repeat things.

words = ["token", "vector", "parse"]
for w in words:
    if len(w) > 5:
        print(w)
# while loop example
count = 0
while count < 3:
    print("Processing...")
    count += 1

4. Functions

Functions are reusable blocks of code.

def clean_text(text):
    return text.lower().replace(".", "").strip()

print(clean_text(" NLP is FUN. "))  # nlp is fun

5. Libraries for NLP

Libraries are like toolkits. They give you pre-built code for common tasks.

# Using nltk to tokenize
import nltk
nltk.download("punkt")
from nltk.tokenize import word_tokenize

sentence = "NLP is cool!"
print(word_tokenize(sentence))

Also explore:

  • spacy: for fast NLP pipelines
  • transformers: for large language models
